Extensions
==========

The following extensions exist that enhance the functionality of GLASS:

- `glass.ext.camb`_ for computing the angular matter power spectrum in the GLASS format with CAMB.
- `glass.ext.pkdgrav`_ for loading PKDGRAV simulations such as the Gower St simulations.

.. _glass.ext.camb: https://pypi.org/project/glass-ext-camb/
.. _glass.ext.pkdgrav: https://pypi.org/project/glass-ext-pkdgrav/
